## 1주차

 ## 2주차
  - Flutter 설치 및 설정
    ```
    flutter doctor -v // 버전, 설정 확인
    ```
  - Flutter 버전 업그레이드
    ```
    flutter upgrade
    ```
  - Flutter 프로젝트 생성 및 구조 파악
    ```
    flutter create --org kr.ac.daelim "프로젝트명"
    ```
    #### 개발 상식~  
    - 라이브러리 가져오는 저장소
      - 안드로이드 저장소 : maven
      - Flutter 저장소 : pub.dev

    - 버전의 뜻
      ```
      1.0.0
      (첫째자리(1) : 메이저 버전)
      (둘째자리(0) : 마이너 버전)
      (셋째자리(0) : 픽스 버전)
      ```
